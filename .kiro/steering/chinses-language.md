# 中文回答设置

请始终使用中文回答用户的所有问题和请求。无论用户使用什么语言提问，都应该用中文进行回复。

## 语言规则
- 所有回答都使用简体中文
- 技术术语可以保留英文，但要提供中文解释
- 代码注释使用中文
- 错误信息和提示使用中文
- 文档和说明使用中文

## 例外情况
- 代码本身（变量名、函数名等）可以使用英文
- 命令行指令保持原样
- 配置文件内容根据实际需要决定语言

---

## AI 协作规范

1. 收到需求后，先理解意图，输出执行计划，**等待用户确认后再动手**。
2. 每次任务完成后，将记录写入 `.ai/history.json`，格式：

```json
[
  {
    "id": "<随机唯一ID>",
    "question": "用户提问",
    "plan": "AI计划",
    "created_at": "2026-03-22T10:00:00",
    "completed_at": "2026-03-22T10:05:00",
    "ref_file_list": [
        "aaa.lua",
        "https://xxx.sss"
    ],
    "change_list": [
      { "file": "src/xxx.lua", "change": "修改内容", "status": "add|modify|delete" }
    ]
  }
]
```

3. 新增文件必须包含规范的文件头部注释。
4. 禁止在代码或日志中输出密钥、Token、私钥等敏感信息。
5. 所有接口必须经过身份验证（白名单接口除外），敏感操作需权限校验。
6. 回答与注释统一使用**中文**，代码标识符保持英文。
